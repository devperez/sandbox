## About BandZilla

This app is not called SandBox but BandZilla.
The main purpose of the app is to help you dicover new bands, artists and albums.
You first have to register and then you can fill in the form. Let's say you like U2, just type it in and the app will return bands that are more or less in the same style. 
Same goes with albums.
Enjoy !
